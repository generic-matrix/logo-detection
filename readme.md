# How to build the Android App

* ```git clone https://github.com/generic-matrix/logo-detection.git```

* cd react-native-app
* npm install
* paste the API key
* react-native run-android


# How To build iOS App

* ```git clone https://github.com/generic-matrix/logo-detection.git```
* cd react-native-app
* npm install
* paste the API key
* react-native run-ios
